# Batchren

A script to rename files with various arguments. Supports Unix style file globbing.

## Requirements
1. Ubuntu 18.04
2. Python 3.5.2
3. Pip
4. natsort (from pypa)

## Instructions
1. Download/clone this repo
2. cd into directory
3. run pip install --user dist/batchren-0.5.2-py3-none-any.whl (should also download natsort)
